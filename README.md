# puristic-deploy

A pnpm + Turbo monorepo of TypeScript packages for solo-dev deployment.

## Getting started

```sh
pnpm install
pnpm build         # turbo build across packages
pnpm test
pnpm libs:watch    # incremental TypeScript build in watch mode
```

> Requires **Node 24** and **pnpm 11**.
